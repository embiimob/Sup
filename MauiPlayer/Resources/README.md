# P2FK Mini Player Resources

This directory contains the application resources for the .NET MAUI mini player.

## Directory Structure

- **Fonts/**: Custom fonts (place OpenSans-Regular.ttf and OpenSans-Semibold.ttf here)
- **Images/**: General images and icons
- **AppIcon/**: Application icon files
- **Splash/**: Splash screen assets
- **Raw/**: Raw assets that are bundled with the app
- **Styles/**: XAML resource dictionaries

## Adding Fonts

Download and add the following fonts to the Fonts directory:
- OpenSans-Regular.ttf
- OpenSans-Semibold.ttf

Or use system fonts by updating MauiProgram.cs

## Adding Icons

Create or add icon files for:
- home.png (Status)
- audio.png (Audio Player)
- video.png (Video Player)
- playlist.png (Playlists)
- search.png (Search)
- settings.png (Setup)

Size recommendations: 1024x1024 PNG with transparency

## App Icon

Replace the default app icon with your custom P2FK icon in the AppIcon directory.

## Splash Screen

Replace the default splash screen in the Splash directory to customize the loading experience.
